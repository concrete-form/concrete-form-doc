## Install

```bash
yarn add @concrete-form/formik
# or with NPM :
npm install @concrete-form/formik
```

Not sure which version to install ? [read this](/docs/how-to-upgrade)

### Requirements

- [**Formik**](https://formik.org/) ; **V2.2+**

## About

`Formik` is the world's most popular open source form library for React. Super simple and easy to learn.
