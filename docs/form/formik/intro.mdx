# Formik

<Header />


## Install

```bash
yarn add @concrete-form/formik
# or with NPM :
npm install @concrete-form/formik
```

Not sure which version to install ? [check this](/docs/getting-started/install#choosing-the-right-version)

### Requirements

- [**Formik**](https://formik.org/) ; **V2.2+**

## About

`Formik` was the most popular library for many years and is still downloaded over 2.5M times per week. If you're starting a new project, [we recommend React-Hook-Form instead](/docs/form/react-hook-form/intro).
