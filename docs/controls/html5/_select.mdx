#### *Options format

:::tip
See [control options](./control-options) for full options specs.
:::

- Options props are forwarded to [`<option />`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/option) elements.
- Group props are forwarded to [`<optgroup />`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/optgroup) elements.
