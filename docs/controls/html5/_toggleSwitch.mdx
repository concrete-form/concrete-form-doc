> a **`boolean`** control that can be toggled **on** or **off**<br />
> **Render: **[`<input type="checkbox" />`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/input/checkbox)

:::tip
Great article about **[Checkbox vs Toggle Switch](https://uxplanet.org/checkbox-vs-toggle-switch-7fc6e83f10b8)**
:::
:::note
In HTML5, a toggle switch is a checkbox with CSS. You need to use our CSS import or create your own. Additionnaly, you can use the available props to customize each HTML elements.
:::

<Props>
  <CommonProps />

  <Prop icon="cf.png" name="label" type="node">

  </Prop>

  <Prop icon="cf.png" name="labelPosition" type={<>"top"<br />| "bottom"<br />| "left"<br />| "right"</>} defaultValue='"right"'>

    <ul>
      <li>the <strong>hidden input</strong> (<code>{` <input /> `}</code>)</li>
      <li>and the <strong>moving slider</strong> (<code>{` <div /> `}</code> element)</li>
    </ul>
  </Prop>

  <Prop icon="html5.png" name="toggleSwitchSliderProps" type="object">
    Forwarded to the toggle switch moving part (<code>{` <div /> `}</code> element)
  </Prop>

  <Prop icon="html5.png" name="{...props}" type="object">
    Forwarded to <code>{' <input /> '}</code> element
  </Prop>

</Props>
