> a **`boolean`** control that can be toggled **on** or **off**<br />
> **Render: **[`<input type="checkbox" />`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/input/checkbox)

:::tip
Great article about **[Checkbox vs Toggle Switch](https://uxplanet.org/checkbox-vs-toggle-switch-7fc6e83f10b8)**
:::
:::note
We render a checkbox with some CSS since there is no toggle switch natively in HTML5. Don't forget to import our CSS file or add your own.
:::
