# Material-UI Controls options

## Multiple choices controls

- [`<Autocomplete />`](./api#autocomplete-)
- [`<Select />`](./api#select-)
- [`<CheckboxesGroup />`](./api#checkboxesgroup-)
- [`<RadiosGroup />`](./api#radiosgroup-)

## Autocomplete
:::caution
`<Autocomplete />` `options` prop is handled by Material-UI. This documentation doesn't apply for that component.<br />
[Check Material-UI documentation](https://mui.com/api/autocomplete/#props).
:::

## `options` prop

**`options`** is expected to be an `array` containing items defined below :

### String
the **key** and the **value** will share the same `string` value provided.

<Demo.StringOptions dense />

Will produce :
```html
<input name="demo" type="radio" value="foo"> foo
<input name="demo" type="radio" value="bar"> bar
<input name="demo" type="radio" value="baz"> baz
```

### Labelled option
Labelled options are defined as object. They can set their value, their label and some props to forward to the control.

| Property | type | description |
| :--- | :--- | :--- |
| label | `node` | Any React node such as string |
| value | `string` | The value of the option |
| props | `object` | Forwarded to the rendered form control |

<Demo.LabelledOptions dense />

Will produce :
```html
<input name="demo" type="checkbox" value="actual-value"> Displayed label
<input name="demo" type="checkbox" value="123"> <strong>I am bold</strong>
<input name="demo" type="checkbox" value="abc" disabled=""> I am disabled
```

### Group
A group has a label, holds other options and cannot be selected.

:::info
Groups are only supported by [`<Select />`](./api#select-).<br />
Groups cannot be nested.
:::

| Property | type | description |
| :--- | :--- | :--- |
| group | `string` | Forwarded to [`<optgroup>`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/optgroup)'s **label** prop or [`<ListSubheader />`](https://mui.com/api/list-subheader/)'s **children**<br />*depending on `native` value* |
| options | `mixed` | Mix of strings and labelled options |
| props | `object` | Forwarded to [`<optgroup>`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/optgroup) or [`<ListSubheader />`](https://mui.com/api/list-subheader/)<br />*depending on `native` value* |
