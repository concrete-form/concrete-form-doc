> a wrapper that helps you create a 2 columns form<br />
> **Render: **[`<Grid />`](https://mui.com/components/grid/) components for the columns

:::info
  By default, both columns are the same size (**width 50%**). On small devices, they are rendered vertically (**width 100%**).
:::
