# Material-UI Advanced layout customization

:::tip
For most projects, you **DON'T** need that level of customization. You should use CSS instead.
:::

## Set custom layout components

Concrete Form allows you to customize every single component used for layout.<br />
These components need to be defined at the `<Form />` level like so:

```jsx
<Form layout={{
  control: ...,
  errors: ...,
  itemLabel: ...,
  itemsGroup: ...,
  label: ...,
  labelledControl: ...,
}}>
  Your form content
</Form>
```

## Available layout components

<CustomLayout hideCode dense />

| layout | description |
| :--- | :--- |
| <div className="cl cl-control">control</div> | Renders **control** and **errors** components |
| <div className="cl cl-errors">errors</div> | Renders the field errors (to be injected in **control**) |
| <div className="cl cl-itemLabel">itemLabel</div> | Renders each item (control + label) in a group (ex: Checkbox or Radio) |
| <div className="cl cl-itemsGroup">itemsGroup</div> | Renders a group of controls (items) |
| <div className="cl cl-label">label</div> | Renders a label inside `LabelledControl` |
| <div className="cl cl-labelledControl">labelledControl</div> | Renders a label and a control |

## Layout components props
Components will receive these props

### control

> **Render: [`<FormControl fullWidth />`](https://mui.com/api/form-control/)**

<Props>
  <Prop name="name" type="string">
    name of the control
  </Prop>

  <Prop name="control" type="node">
    Control component
  </Prop>

  <Prop name="errors" type="node">
    Errors component. (result of <div className="cl cl-errors">errors</div> layout)
  </Prop>

  <Prop name="{...props}" type="object">
    All props provided with <code> formControlProps </code>. (see <a href="https://mui.com/api/form-control/#props" target="_blank">Material-UI doc</a>)
  </Prop>
</Props>

:::caution
On the example above, notice the missing pink line between the **text control** and the **error**. The reason is Material-UI will handle the error inside the [TextField](https://mui.com/components/text-fields/) component. If that's the case, `errors` will be empty here since it's already included in `control`.
:::

### errors

<Props>
  <Prop name="name" type="string">
    name of the control
  </Prop>

  <Prop name="errors" type="string[]">
    array of translated errors
  </Prop>
</Props>

### itemLabel

> **Render: [`<FormControlLabel />`](https://mui.com/api/form-control-label/)**

<Props>
  <Prop name="name" type="string">
    name of the control
  </Prop>

  <Prop name="control" type="node">
    control element (checkbox, radio, toggle switch, etc)
  </Prop>

  <Prop name="label" type="node">
    label of the control
  </Prop>

  <Prop name="labelPosition" type={<>"top"<br />| "bottom"<br />| "left"<br />| "right"</>}>
    position of the label
  </Prop>
</Props>

### itemGroup

> **Render: [`<FormGroup />`](https://mui.com/api/form-group/)**

<Props>
  <Prop name="name" type="string">
    name of the control
  </Prop>

  <Prop name="items" type="node">
    result of multiple <div className="cl cl-itemLabel">itemLabel</div><br />
    <strong>Note:</strong> this is a single node and <strong>not</strong> an array
  </Prop>

  <Prop name="label" type="node">
    label of the control
  </Prop>

  <Prop name="orientation" type={<>"vertical" | "horizontal"</>}>
    orientation of the items
  </Prop>
</Props>

### label

> **Render: [`<FormLabel />`](https://mui.com/api/form-label/)**

<Props>
  <Prop name="label" type="node">
    label to render
  </Prop>

  <Prop name="htmlFor" type="string | undefined">
    "<a href="https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/for" target="_blank"><strong>for</strong></a>" value to provide to the label (<em>if available</em>)
  </Prop>

  <Prop name="{...props}" type="object">
    All extra props provided to <code> LabelledControl </code>. (see <a href="https://mui.com/api/form-label/#props" target="_blank">Material-UI doc</a>)
  </Prop>
</Props>

### labelledControl

> **Render: [`<Grid />`](https://mui.com/components/grid/)** components

<Props>
  <Prop name="control" type="node">
    result of <div className="cl cl-control">control</div>
  </Prop>

  <Prop name="label" type="node">
    result of <div className="cl cl-label">label</div>
  </Prop>

  <Prop name="labelPosition" type={<>"top"<br />| "bottom"<br />| "left"<br />| "right"</>}>
    position of the label (relative to the control)
  </Prop>

  <Prop name="mainGridProps" type="object">
    <strong>main</strong> <a href="https://mui.com/components/grid/" target="_blank"><code>{'<Grid container />'}</code></a> component
  </Prop>

  <Prop name="labelGridProps" type="object">
    <strong>label</strong> <a href="https://mui.com/components/grid/" target="_blank"><code>{'<Grid item />'}</code></a> component
  </Prop>

  <Prop name="controlGridProps" type="object">
    <strong>control</strong> <a href="https://mui.com/components/grid/" target="_blank"><code>{'<Grid item />'}</code></a> component
  </Prop>
</Props>
