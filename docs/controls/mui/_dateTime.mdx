> a date, time or datetime picker. This control handles a **[date](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Date) object**. <br />
> **Render: ** [`<DatePicker />`](https://mui.com/components/date-picker), [`<TimePicker />`](https://mui.com/components/time-picker), or [`<DateTimePicker />`](https://mui.com/components/date-time-picker), 

:::tip
Don't forget to install `@mui/x-date-pickers` and to follow the [setup instructions provided by Material-UI](https://mui.com/components/date-picker/#requirements) in order to use the Pickers components.
:::
