## Install

```bash
yarn add @concrete-form/html5
# or with NPM :
npm install @concrete-form/html5
```

Not sure which version to install ? [check this](/docs/getting-started/install#choosing-the-right-version)

## Dependencies

**tailwindcss**

If `Tailwind CSS` is not already installed on your projet, you'll need to install and configure it. [Get started with Tailwind CSS](https://tailwindcss.com/docs/installation/framework-guides).

**@tailwindcss/forms**

We strongly recommend the official form plugin.
> Provides a basic reset for form styles that makes form elements easy to override with utilities.

[Github link](https://github.com/tailwindlabs/tailwindcss-forms)
