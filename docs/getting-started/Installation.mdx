# Installation

:::info
You need to install **2 packages** to get started. (Form + UI)
:::

## 1) Form

**React Hook Form** (`react-hook-form` required)

```bash
yarn add @concrete-form/react-hook-form
```

**Formik** (`formik` required)

```bash
yarn add @concrete-form/formik
```

## 2) UI

**HTML5 (Highly customizable)**

```bash
yarn add @concrete-form/html5
```

**Material-UI** (`mui` required)

```bash
yarn add @concrete-form/mui
```

**Materialize** (`materialize` required)
```bash
yarn add @concrete-form/materialize
```
